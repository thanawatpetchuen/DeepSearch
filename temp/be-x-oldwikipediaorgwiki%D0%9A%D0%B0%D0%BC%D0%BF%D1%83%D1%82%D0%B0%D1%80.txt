<p><b>Кампу́тар</b> — вылічальная <a href="/wiki/%D0%9C%D0%B0%D1%88%D1%8B%D0%BD%D0%B0" title="Машына">машына</a> для апрацоўкі <a href="/wiki/%D0%86%D0%BD%D1%84%D0%B0%D1%80%D0%BC%D0%B0%D1%86%D1%8B%D1%8F" title="Інфармацыя">зьвестак</a>, паводле зададзеных інструкцыяў. Сучасны кампутар складаецца з абсталяваньня, якое ўяўляе фізычныя часткі кампутара, як то <a href="/wiki/%D0%A6%D1%8D%D0%BD%D1%82%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B_%D0%BF%D1%80%D0%B0%D1%86%D1%8D%D1%81%D0%B0%D1%80" title="Цэнтральны працэсар">працэсар</a>, <a href="/wiki/%D0%9A%D0%BB%D1%8F%D0%B2%D1%96%D1%8F%D1%82%D1%83%D1%80%D0%B0" title="Клявіятура">клявіятура</a>, <a href="/wiki/%D0%9C%D0%B0%D0%BD%D1%96%D1%82%D0%BE%D1%80" title="Манітор">манітор</a> і гэтак далей, і <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B3%D1%80%D0%B0%D0%BC%D0%BD%D0%B0%D0%B5_%D0%B7%D0%B0%D0%B1%D0%B5%D1%81%D1%8C%D0%BF%D1%8F%D1%87%D1%8D%D0%BD%D1%8C%D0%BD%D0%B5" title="Праграмнае забесьпячэньне">праграмнага забесьпячэньня</a>, як то <a href="/wiki/%D0%90%D0%BF%D1%8D%D1%80%D0%B0%D1%86%D1%8B%D0%B9%D0%BD%D0%B0%D1%8F_%D1%81%D1%8B%D1%81%D1%82%D1%8D%D0%BC%D0%B0" title="Апэрацыйная сыстэма">апэрацыйнай сыстэмы</a> і <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B3%D1%80%D0%B0%D0%BC%D0%BD%D0%B0%D0%B5_%D0%B7%D0%B0%D0%B1%D0%B5%D1%81%D1%8C%D0%BF%D1%8F%D1%87%D1%8D%D0%BD%D1%8C%D0%BD%D0%B5" title="Праграмнае забесьпячэньне">прыкладньняў</a>).</p>
<p>Кампутары маюць некалькі фізычных формаў. Першыя прылады, якія нагадваюць сучасны кампутар, зьявіліся ў сярэдзіне XX стагодзьдзя, прыкладна ў 1940—1945 гады, аднак машыны, падобныя на кампутары, існавалі і раней. Першыя электронныя кампутары мелі памер вялікіх памяшканьняў, спажывалі электрычнай энэргіі як некалькі сотняў сучасных <a href="/wiki/%D0%9F%D1%8D%D1%80%D1%81%D0%B0%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B_%D0%BA%D0%B0%D0%BC%D0%BF%D1%83%D1%82%D0%B0%D1%80" title="Пэрсанальны кампутар">пэрсанальных кампутараў</a>.</p>
<p>У цяперашні час кампутары выкарыстоўваюцца практычна ва ўсіх абласьцях чалавечай дзейнасьці.</p>

<h2><span id=".D0.AD.D1.82.D1.8B.D0.BC.D0.B0.D0.BB.D1.91.D0.B3.D1.96.D1.8F"></span><span class="mw-headline" id="Этымалёгія">Этымалёгія</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%BC%D0%BF%D1%83%D1%82%D0%B0%D1%80&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Рэдагаваць сэкцыю «Этымалёгія»">рэдагаваць</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%BC%D0%BF%D1%83%D1%82%D0%B0%D1%80&amp;action=edit&amp;section=1" title="Рэдагаваць сэкцыю «Этымалёгія»">рэдагаваць крыніцу</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Слова <i>кампутар</i> мае паходжаньне ад <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%BC%D0%BE%D0%B2%D0%B0" title="Ангельская мова">ангельскага</a> <i>to compute</i>, <i>computer</i>, якія перакладаюцца як «вылічваць», «вылічальнік» (ангельскае слова ў сваю чаргу паходзіць з лацінскага <i>computo</i> — «вылічваю»). Першапачаткова ангельскае слова тлумачылі як чалавека, які робіць <a href="/wiki/%D0%90%D1%80%D1%8B%D1%82%D0%BC%D1%8D%D1%82%D1%8B%D0%BA%D0%B0" title="Арытмэтыка">арытмэтычныя</a> вылічваньні з дапамогай ці без мэханічных прыладаў. Потым ужо слова тлумачылі як самі прылады, але ж у сучасных кампутарах трываюць і іншыя працэсы акрамя вылічваньняў.</p>
<p>Упершыню тлумачэньне слова «кампутар» зьявілася ў 1897 годзе ў <a href="/w/index.php?title=%D0%9E%D0%BA%D1%81%D1%84%D0%B0%D1%80%D0%B4%D0%B7%D0%BA%D1%96_%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D1%96_%D1%81%D0%BB%D0%BE%D1%9E%D0%BD%D1%96%D0%BA&amp;action=edit&amp;redlink=1" class="new" title="Оксфардзкі ангельскі слоўнік (старонка не існуе)">Оксфардзкім ангельскім слоўніку</a>. Тады слова тлумачылася як мэханічная прылада, але ў 1946 годзе ў слоўніку былі зьмешчаны дадаткі, якія дазвалялі абасобіць паняцьця лічбовага, аналягавага й электроннага кампутараў.</p>
<h2><span id=".D0.93.D1.96.D1.81.D1.82.D0.BE.D1.80.D1.8B.D1.8F"></span><span class="mw-headline" id="Гісторыя">Гісторыя</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%BC%D0%BF%D1%83%D1%82%D0%B0%D1%80&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Рэдагаваць сэкцыю «Гісторыя»">рэдагаваць</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%BC%D0%BF%D1%83%D1%82%D0%B0%D1%80&amp;action=edit&amp;section=2" title="Рэдагаваць сэкцыю «Гісторыя»">рэдагаваць крыніцу</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.9D.D0.B5.D0.BA.D0.B0.D0.BC.D0.BF.D1.83.D1.82.D0.B0.D1.80.D0.BD.D0.B0.D0.B5_.D0.BF.D0.B0.D0.BA.D0.B0.D0.BB.D0.B5.D0.BD.D1.8C.D0.BD.D0.B5"></span><span class="mw-headline" id="Некампутарнае_пакаленьне">Некампутарнае пакаленьне</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%BC%D0%BF%D1%83%D1%82%D0%B0%D1%80&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Рэдагаваць сэкцыю «Некампутарнае пакаленьне»">рэдагаваць</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%BC%D0%BF%D1%83%D1%82%D0%B0%D1%80&amp;action=edit&amp;section=3" title="Рэдагаваць сэкцыю «Некампутарнае пакаленьне»">рэдагаваць крыніцу</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>3000 гадоў да н. э. — у <a href="/w/index.php?title=%D0%A1%D1%82%D0%B0%D1%80%D0%B0%D0%B6%D1%8B%D1%82%D0%BD%D1%8B_%D0%91%D0%B0%D0%B1%D1%96%D0%BB%D1%91%D0%BD&amp;action=edit&amp;redlink=1" class="new" title="Старажытны Бабілён (старонка не існуе)">Старажытным Бабілёне</a> вынайшлі першыя лічыльнікі — <a href="/wiki/%D0%90%D0%B1%D0%B0%D0%BA" title="Абак">абак</a>.</li>
<li>500 гадоў да н. э. — у <a href="/wiki/%D0%9A%D1%96%D1%82%D0%B0%D0%B9" title="Кітай">Кітаі</a> зьявіўся больш «сучасны» варыянт <a href="/wiki/%D0%90%D0%B1%D0%B0%D0%BA" title="Абак">абака</a> з косткамі на дроце.</li>
</ul>
